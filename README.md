Python-разработчик, специализируюсь на создании Telegram-ботов и веб-приложений.
Моё знакомство с программированием началось в университете, где я освоил основы Python и постепенно углубился в разработку ботов для Telegram.

За два года я прошёл путь от простых скриптов до полноценных ботов с базами данных, inline-клавиатурами и интеграциями. В последнее время активно осваиваю TypeScript и разработку Telegram Mini Apps.

- 🎓 Изучаю программирование в рамках университетской программы
- 🤖 Специализируюсь на Telegram-ботах на базе aiogram 3 и Telegraf
- 🌐 Разрабатываю Telegram Mini Apps на React + TypeScript
- 🛠 Работаю с PostgreSQL, Redis, Docker, Railway, Vercel, Render
- 📂 Собираю портфолио концепт-проектов
